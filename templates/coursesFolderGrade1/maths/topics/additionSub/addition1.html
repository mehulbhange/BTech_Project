Addition using a number line

Kid should watch this lesson carefully and parents should teach below points to their child

Counting on number line is the playful method through which the child is learning addition of numbers. Ask math, explains you the addition of numbers using colorful number line that makes the children more interested. Addition means plus. The sign of addition is + . Addition of the numbers on number line this method is used only for small numbers. For smaller grades the number line starts with zero. Number line is the line which starts from zero and has numbers 1,2,3 â€¦ on it at equal distances. It is used in addition,subtraction and multiplication because you can jump from one number to another number.

Steps for addition on number line
  • Draw a number line.
  • According to the question, from zero jump the first number.
  • Mark the first number.
  • From the first number jump second number.
  • Mark it. That gives you the answer.
Example
  • Draw a number line. Say up to 15.
  • From zero jump to 5. Mark it.
  • From 5 jump 7 number. Mark it .
  • So the answer of 5 + 7 = 12 .
